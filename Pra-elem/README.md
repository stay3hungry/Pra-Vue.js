# elem

> A Vue.js project

## Build Setup

``` bash
# install dependencies
cnpm install

# serve with hot reload at localhost:1018
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
技术：
1、 html + css + js
2、 webpack配置
3、 vue组件引用、数据传递
4、 vue-resource
5、 布局
